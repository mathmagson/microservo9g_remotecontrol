#This project is a Radio Transmitter controlling an Micro Servo using an ARM microcontroller.

##Components:

###RADIO TRANSMITTER: TURNIGY 9XR
###MICROCONTROLLER: ARM TM4C123GH6PM
###SERVO: MICRO SERVO 9g
